The Pomodoro Technique was created by Francesco Cirillo to compartmentalize focus into discrete segments of time. The goal is improved productivity.

The technique can be used in the following manner:

  1. Decide on a task to accomplish
  2. Start the 25 minute timer
  3. Work without concern for time
    • If you stray away from your task, restart the timer
  4. When the timer goes off
    • If you've done less than 4 Pomodoros, take a 5 minute break-this timer will start automatically
    • If you've done 4 Pomodoros, take a 30 minute break-this timer will also start on its own
  5. Lather, rinse, repeat!
